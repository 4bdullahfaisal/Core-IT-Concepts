## **2. REVIEW CORE IT CONCEPTS**

Let me give you **quick answers** for each:

### **Programming Basics**
- **Conditions:** `if`, `else`, `switch`
- **Loops:** `for`, `while` (repeat code)
- **Functions:** Reusable blocks of code
- **OOP:** Classes, objects, inheritance, encapsulation

### **Data Structures (Simple definitions)**
| Structure | What it is |
|-----------|-------------|
| Array | Fixed-size list of items |
| Linked List | Chain of connected items |
| Stack | Last-In-First-Out (like undo) |
| Queue | First-In-First-Out (like printer queue) |

### **SQL Basics**

```sql
SELECT * FROM users WHERE id = 1;
INSERT INTO users (name) VALUES ('John');
UPDATE users SET name = 'Ali' WHERE id = 1;
DELETE FROM users WHERE id = 1;
```

### **Networking Basics**

| Term | Meaning |
|------|---------|
| IP Address | Device's unique address on network |
| DNS | Converts domain (google.com) to IP |
| HTTP | Web traffic (port 80) |
| HTTPS | Secure web traffic (port 443) |

### **Operating Systems Basics**

| Term | Meaning |
|------|---------|
| Process | Running program |
| Memory | RAM storage |
| File System | How files are organized on disk |

## **3. PRACTICE EXPLAINING YOUR PROJECTS**

Use the **STAR method**:

| Letter | Meaning | Example |
|--------|---------|---------|
| **S** | Situation | "We needed to deploy a web app securely" |
| **T** | Task | "My role was to set up the server" |
| **A** | Action | "I installed Apache, configured firewall, created users" |
| **R** | Result | "The app is running and secure" |

**Your project to practice:**

> *"I built a secure multi-user application server with Apache, user management, firewall, and automated backups."*

## **5. PROFESSIONAL BEHAVIOUR & COMMUNICATION**

### **Appearance**
- **Wear:** Shirt + trousers (clean, ironed)
- **No:** T-shirts, jeans, sports shoes

### **Body Language**
| Do | Don't |
|----|-------|
| Maintain eye contact | Look at the floor |
| Sit up straight | Slouch |
| Nod to show listening | Cross arms |
| Smile naturally | Look nervous/fidget |

### **Communication**
- Speak **clearly** and **slowly**
- Listen **before** answering
- No slang ("yeah", "gonna", "ain't")
- Use "Sir/Ma'am" if appropriate

## **9. DO'S & DON'TS**

| Do | Don't |
|----|-------|
